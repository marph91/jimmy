# Convert from Reflect to Markdown

## General Information

- [Website](https://reflect.app/)
- Typical extension: `.json`

## Instructions

1. Export as described [at the website](https://reflect.academy/import-export-backups)
    1. Select format "Reflect JSON"
3. [Install Jimmy](../index.md#installation)
4. Convert to Markdown. Example: `jimmy-linux cli reflect-20251219-2329-v1.0.json --format reflect`
5. [Import to your app](../import_instructions.md)
